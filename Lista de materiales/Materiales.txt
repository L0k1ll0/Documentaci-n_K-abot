Lista de Materiales para K-abot

Componentes Electrónicos:
1. Arduino Nano
2. Shield para Arduino Nano
3. Sensor ultrasónico HC-SR04
4. Servomotores:
   - 1 Servo para la base
   - 2 Servos para el brazo
5. Cables de conexión (jumpers)

Componentes Impresos en 3D:
1. Base del brazo robótico
2. Soporte para servomotores
3. Eslabones y uniones del brazo

Otros Materiales:
1. Tornillos y tuercas** para ensamblar las partes impresas
2. Fuente de alimentación** para Arduino Nano
3. Pegamento** (opcional, para fijar algunas partes si es necesario)

Herramientas:
1. Impresora 3D** para las partes impresas
2. Destornillador** para ensamblar el brazo robótico

Con estos materiales y herramientas, podrás armar y programar tu K-abot siguiendo las instrucciones y el código proporcionado.
